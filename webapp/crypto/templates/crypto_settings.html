How to generate read-only keys
  1. Login to binance.com → Profile → API Management
  2. Click "Create API" → "System generated"
  3. Name it: stock-platform-readonly
  4. Permissions: ✓ Enable Reading only. Leave trading + withdrawals OFF.
  5. Optional: IP-restrict to your Mac's public IP
  6. Copy both API Key + Secret (Secret shown ONCE)
